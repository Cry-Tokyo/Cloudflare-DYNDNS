# Cloudflare-DYNDNS

Script that uses Cloudflare's API to implement DDNS(Dynamic Domain Name Service)

### Installing


Clone Cloudflare-DYNDNS.sh:

```
git clone https://github.com/MCgit2024/Cloudflare-DYNDNS-script.git DDNS
cd DDNS
chmod 700 Dyn-DNS.sh

```
